# Vue.js Date-Time Picker

## Install

```bash
npm install vuejs-datetime-picker
```

```javascript
import { Datetimepicker, Datepicker, Timepicker } from 'vuejs-datetime-picker'

export default {
  // ...
  components: {
    Datetimepicker,
  },
}
```
